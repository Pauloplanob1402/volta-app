# **App Name**: Volta - Find Your Way Back

## Core Features:

- Phrase Display: Display a single, encouraging phrase from a pre-defined list.
- Phrase Refresh: Allow the user to request a new phrase with a button click.
- Phrase List: Maintain an internal list of phrases focused on resisting social media urges.
- PWA Support: Enable the app as a Progressive Web App for installability and offline access.

## Style Guidelines:

- Primary color: Soft Blue (#A0D2EB) for a calming and peaceful feel.
- Background color: Very light gray (#F5F5F5) to keep the focus on the phrase.
- Accent color: Light Green (#90EE90) for the button, providing a subtle encouragement without being overwhelming.
- Body and headline font: 'Inter' for a clean, modern and readable style, ensuring clarity and simplicity.
- Simple, centered layout optimized for mobile. Prominent display of the phrase with the refresh button below.
- Subtle fade-in animation when a new phrase is displayed.
